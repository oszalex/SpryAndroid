# README

## Install

> . venv/bin/activate  
> sudo pip install -r requirements.txt  


## Commands

`python manager.py clean_db` deletes all tables

`python manager.py seed` seed db with dummy values

`python manager.py reset` clear and seed db with dummy values

## Technologies

 * relational DB (MySQL)
 * ORM (SQLalchemy)
 * Python
